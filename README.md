# loan-vs-voo

Small **Streamlit** app: compare **buying a home** vs **renting + investing in VOO** (loan, HOA, property tax, marginal rate on mortgage interest, rent growth, etc.—all sliders).

**Live:** <https://loan-vs-voo-pt4vamuzm4ahrdxxgryspj.streamlit.app/>

**Local**

```bash
pip install -r requirements.txt
streamlit run app.py
```
